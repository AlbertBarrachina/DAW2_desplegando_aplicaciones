class calculadora.calculadora[source]¶

Bases: object

A simple calculator class for basic arithmetic operations.

Methods: - sumar(x, y): Add two numbers. - restar(x, y): Subtract two numbers. - multiplicar(x, y): Multiply two numbers. - dividir(x, y): Divide two numbers.

dividir(x, y)[source]¶

Divide two numbers and return the result.

This method performs division between two numbers and returns the result. It also handles division by zero, raising a ValueError if y is 0.

Parameters:

  • x (float or int) – The dividend. It can be a float or an integer.

  • y (float or int) – The divisor. It can be a float or an integer.

Returns:

The result of the division. The return type matches the type of the operands.

Return type:

float or int

Raises:

ValueError – If the divisor (y) is 0, a ValueError is raised to prevent division by zero.
